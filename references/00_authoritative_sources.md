# 权威依据（Reference Pack）

本 Skill 面向 **中国发明专利**的交底书撰写与初步检索/新颖点提炼。

## 1) 新颖性/创造性/实用性（专利法第22条）
- 专利法第22条规定：授予专利权的发明和实用新型应当具备新颖性、创造性、实用性，并给出了“新颖性”的定义（不属于现有技术、无抵触申请）。  
- CNIPA 在线文本（2020年修正，含第22/26条）：https://www.cnipa.gov.cn/art/2020/11/23/art_97_155167.html

## 2) 申请文件与说明书“清楚、完整”（专利法第26条）
- 发明申请应提交：请求书、说明书及其摘要、权利要求书等；说明书应当清楚、完整；权利要求应以说明书为依据并清楚简要限定保护范围。  
- CNIPA 在线文本：https://www.cnipa.gov.cn/art/2020/11/23/art_97_155167.html

## 3) 说明书章节结构与写作规范（实施细则第20条，2023修订）
- 说明书应包含：技术领域、背景技术、发明内容、附图说明、具体实施方式，并按规定顺序撰写；用词规范，语句清楚，不得使用商业性宣传用语等。  
- CNIPA 实施细则：https://www.cnipa.gov.cn/art/2023/12/21/art_98_189197.html

## 4) 专利审查指南（2023）
- 用于进一步理解审查口径与写作方法（非本 Skill 的法律结论替代）。  
- CNIPA 发布页面（含下载）：https://www.cnipa.gov.cn/art/2023/12/21/art_2790_189500.html

## 5) 国际写作方法（WIPO）
- WIPO Patent Drafting Manual（2023）：对说明书与权利要求撰写方法学有帮助。  
- PDF：https://www.wipo.int/edocs/pubdocs/en/wipo-pub-867-23-en-wipo-patent-drafting-manual.pdf
