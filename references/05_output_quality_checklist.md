# 输出质量检查清单（QA Checklist）

## 交底书结构
- [ ] 技术领域
- [ ] 背景技术（现有方案 + 缺陷）
- [ ] 技术问题（1–3条）
- [ ] 技术方案（总体 + 关键特征 F1..Fn）
- [ ] 有益效果（无数据则“待补充 + 验证建议”）
